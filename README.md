# 🚀 Online Quiz System

A full-stack web application built using Laravel (PHP) and PostgreSQL that allows users to take quizzes, track performance, and view rankings through an interactive dashboard.

---

## ✨ Features

### 👨‍🎓 User Panel
- User Authentication (Login/Register)
- Attempt quizzes
- Real-time score calculation
- View result with:
  - Score
  - Percentage
  - Rank
  - Performance
- Leaderboard system
- Download certificate

---

### 🛠️ Admin Panel
- Manage Quizzes
- Manage Subjects & Categories
- Manage Questions
- Manage Students
- View Results
- Full CRUD operations

---

## 🧠 Tech Stack

- Backend: Laravel (PHP)
- Database: PostgreSQL
- Frontend: Blade + Tailwind CSS
- Authentication: Laravel Auth
- Version Control: Git & GitHub
