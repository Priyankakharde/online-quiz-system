🚀 Online Quiz System

A full-stack web application built with Laravel (PHP) and PostgreSQL that allows users to take quizzes, track performance, and view rankings through an interactive dashboard.

📌 Features

👨‍🎓 User Side

User authentication (login/register)
Interactive quiz system
Real-time score calculation
Performance analysis (score, percentage, rank)
Leaderboard system
Downloadable certificate

🛠 Admin Panel

Manage quizzes, subjects, and questions
Manage students
View and control results
Full CRUD operations

🧠 Tech Stack

Backend: PHP (Laravel)
Database: PostgreSQL
Frontend: Blade + Tailwind CSS
Authentication: Laravel Auth
Version Control: Git & GitHub
